### Driver Flag options
- -i <filename>:
  - The template format input file is the file **_driver_input_**  
- -nor <boolean>:
  - **_False_** reduce is used and this is the default.
  - **_True_** reduce is not used.

### Log file
The driver log file is names **_newfile.log_**
